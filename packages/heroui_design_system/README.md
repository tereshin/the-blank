# heroui_design_system

HeroUI design system package for Flutter.

## Installation

```yaml
dependencies:
  heroui_design_system:
    git:
      url: https://github.com/tereshin/the-blank.git
      path: packages/heroui_design_system
```

## Usage

```dart
import 'package:heroui_design_system/design_system.dart';
```

## Included

- HeroUI widgets
- Typography tokens
- SVG icon loader and icon assets
- Inter font assets
